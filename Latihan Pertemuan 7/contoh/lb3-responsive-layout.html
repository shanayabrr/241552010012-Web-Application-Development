LB 3 — Responsive Layout

Resize jendela browser untuk melihat perubahan breakpoint.

Soal 1 — Mobile First dengan Media Query
Buat grid card yang berubah jumlah kolomnya: 1 kolom (mobile) → 2 kolom (≥640px) → 4 kolom (≥1024px).

Lihat kunci jawaban 
#s1 {
  grid-template-columns: 1fr;
}
@media (min-width: 640px) {
  #s1 { grid-template-columns: repeat(2, 1fr); }
}
@media (min-width: 1024px) {
  #s1 { grid-template-columns: repeat(4, 1fr); }
}
Soal 2 — auto-fit + minmax() (tanpa media query)
Gunakan repeat(auto-fit, minmax(...)) agar kolom terbentuk otomatis. Lebar minimum kolom 180px, maksimum 1fr.

Lihat kunci jawaban 
#s2 {
  grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
}
Soal 3 — Layout Artikel Responsif
Mobile: semua area tersusun vertikal.
≥768px: header atas, artikel (3fr) di kiri + iklan (1fr) di kanan, footer bawah. Gunakan grid-template-areas.
Header
Artikel
Iklan
Footer
Lihat kunci jawaban 
/* Mobile */
#s3 {
  grid-template-areas:
    "header"
    "artikel"
    "iklan"
    "footer";
  grid-template-columns: 1fr;
}

/* ≥768px */
@media (min-width: 768px) {
  #s3 {
    grid-template-areas:
      "header  header"
      "artikel iklan"
      "footer  footer";
    grid-template-columns: 3fr 1fr;
  }
}

#s3 .header  { grid-area: header; }
#s3 .artikel { grid-area: artikel; }
#s3 .iklan   { grid-area: iklan; }
#s3 .footer  { grid-area: footer; }
