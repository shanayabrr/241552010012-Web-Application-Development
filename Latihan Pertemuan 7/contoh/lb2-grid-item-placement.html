Soal 3 — grid-template-areas
Lengkapi grid-template-areas di container #s3 dan grid-area di setiap item agar membentuk layout: header (penuh), sidebar (kiri), content (kanan 3/4), footer (penuh).
Header
Sidebar
Content
Footer
Lihat kunci jawaban 
#s3 {
  grid-template-areas:
    "header  header  header  header"
    "sidebar content content content"
    "footer  footer  footer  footer";
}
#s3 .header  { grid-area: header; }
#s3 .sidebar { grid-area: sidebar; }
#s3 .content { grid-area: content; }
#s3 .footer  { grid-area: footer; }
